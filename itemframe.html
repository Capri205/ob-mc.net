OB-Build Item frame maps:

OB Build server uses a lot of maps to enhance the look of the world. These are built using a handly little utility called MC Map Item Tool from djfun. It can be found here https://mc-map.djfun.de/

These maps are placed atop of item frames. There are single 1x1 (1 map), 1x2 and 2x1 (2 maps), 2x2 (4 maps) and so on. The table below for each item shows you how many maps you need.


Below are the map ID numbers to use with the give command to get the maps, where is your minecraft name and is the ID of the map from the list below: 
/give yourid filled_map 1 {map:mapid}

We are happy to load your map files into the server *after* you have tested your map files yourself. Email the zip file produced by the program above to obmcnet@gmail.com. Use 990 for the ID and we will rename the files to appropriate ID and update the list below with the new ID's.
